In the early Twentieth Century Ireland, St. John D Seymour embarked on a remarkable experiment in crowd sourcing, designed to gather a collection of verified "true" Irish ghost stories. The goal of this site is to try to find out more about those stories, and if possible, update them for the Twenty First Century. Please tell us if you can add anything.
Do these ghosts still haunt these places or have they disappeared? Perhaps they've been replaced by more modern phantoms?
